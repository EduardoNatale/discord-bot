# discord-bot

It's is an API

Para usar essa aplicação é necessário criar na raiz do projeto um .env

### Para a API do twitter

```
apikey
apikeysecret
accesstoken
accesstokensecret
```

### Para a API do discord

```
discordtoken
```

### Para as APIs da HG

```
hgkey
```

### Para rodar

`npm run dev`
