# discord-bot

To set up this project is necessary to create a .env

### Twitter's API

```
apikey
apikeysecret
accesstoken
accesstokensecret
```

### Discord's API

```
discordtoken
```

### HG's APIs

```
hgkey
```

`npm run dev`
